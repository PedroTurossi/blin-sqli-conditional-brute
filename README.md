# Brute-force de Blind-SQLi com respostas condicionais
Esse foi um código desenvolvido durante testes nos laboratórios da Port Swigger de SQLi (SQL Injection). Mais especificamente no laboratório "Blind SQL injection with conditional responses", onde existia uma leve diferença no site se a consulta acontecia ou não.
Logo, foi possivel implementar uma lógica que pegava pedaço por pedaço da informação e conseguir tudo o que era necessário para concluir o laboratório automaticamente. Nesse caso, o objetivo era pegar a senha do usuário "Administrator".
